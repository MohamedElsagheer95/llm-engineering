
<div class="box">
  <h1>🚀 AI-Powered Marketing Brochure Maker</h1>
  <p>
    An intelligent AI system that automatically transforms any company website into a
    high-quality, structured marketing brochure using Large Language Models and web intelligence.
  </p>
</div>

<div class="box">
  <h2>✨ Overview</h2>
  <p>
    This system extracts website content, filters relevant business information, and generates
    a professional marketing brochure using AI. It eliminates manual copywriting and produces
    ready-to-use marketing material in seconds.
  </p>
</div>

<div class="box">
  <h2>🎯 Key Features</h2>
  <ul>
    <li>🌐 Automatic website crawling and link extraction</li>
    <li>🔗 AI-based filtering of relevant pages</li>
    <li>🤖 Multi-model support (Gemini + Ollama)</li>
    <li>🧠 Smart content aggregation</li>
    <li>✨ High-quality marketing brochure generation</li>
    <li>⚡ Fully automated pipeline</li>
  </ul>
</div>

<div class="box">
  <h2>🧠 System Architecture</h2>
  <pre>
Website URL
   ↓
Extract all links
   ↓
AI filters relevant pages
   ↓
Scrape selected pages
   ↓
Aggregate content
   ↓
Generate brochure using LLM
   ↓
Final output
  </pre>
</div>

<div class="box">
  <h2>🔄 Pipeline Workflow</h2>
  <ol>
    <li>Input website URL</li>
    <li>Scrape all website links</li>
    <li>AI selects relevant marketing pages</li>
    <li>Extract page content</li>
    <li>Combine into structured context</li>
    <li>Generate final brochure using LLM</li>
  </ol>
</div>

<div class="box">
