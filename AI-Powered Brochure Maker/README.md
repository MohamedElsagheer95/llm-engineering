# 🚀 AI-Powered Marketing Brochure Maker

An intelligent AI system that automatically generates high-quality, persuasive marketing brochures from any company website using LLMs, web scraping, and content intelligence.

---

## ✨ Overview

The **AI-Powered Marketing Brochure Maker** extracts structured information from websites, filters the most relevant business pages, and transforms them into a polished, customer-ready marketing brochure.

It eliminates manual copywriting and turns raw web content into **professional marketing material in seconds**.

---

## ⚙️ Key Features

- 🌐 Automatic website scraping
- 🔗 AI-based link filtering (removes noise like login, policies, admin pages)
- 🤖 Multi-model support (Gemini + Ollama)
- 📄 Structured brochure generation
- ✨ Marketing-focused content rewriting
- 🧠 Smart content aggregation from multiple pages
- ⚡ Clean, production-ready output

---

## 🧠 How It Works

### 1. Input Website URL
Provide any company website.

### 2. Link Extraction
The system scrapes all available links from the website.

### 3. AI Link Filtering
LLM selects only relevant marketing pages such as:
- About Us
- Products / Services
- Pricing
- Careers
- Blog / Case Studies

### 4. Content Extraction
Relevant pages are scraped and combined into a unified context.

### 5. AI Brochure Generation
A language model transforms the content into a structured brochure.

---

## 📦 Output Structure

The generated brochure includes:

- 🚀 Company Name  
- ✨ Tagline  
- 💡 About Us  
- 📦 Products / Services  
- ⚡ Key Features / Advantages  
- 🏆 Why Choose Us  
- 📞 Call to Action (CTA)
